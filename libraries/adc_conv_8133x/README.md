# BU-Act ADC Conversion Library

## General
* Customer: Melexis
* Product(s): MLX81330/32
* Description: Libraries

## Getting started
Library providing conversion functions for the adc module, these functions will convert
adc measurements into SI units as documented in the [MLX81330 calibration user manual](https://softdist.melexis.com/custassetsJRightFrame?assetname=MLX81330_Calibration_Document) and
[MLX81332 calibration user manual](https://softdist.melexis.com/custassetsJRightFrame?assetname=MLX81332_Calibration_Document).

## Installation

Add *adc_conv_8133x* to the BU_LIBS list in the file Makefile.srcs.mk located in the application source folder.

```
BU_LIBS += adc_conv_8133x
```
